# 💰 Student Finance App
Aplikasi ini dibuat buat membantu mahasiswa mencatat pemasukan, pengeluaran, saldo, melihat statistik keuangan, sampai membuat laporan keuangan. Jadi nggak ada lagi alasan uang bulanan hilang secara misterius 🗿

---

## 🚀 Cara Clone Project

Clone dulu repository ini:

```bash
git clone https://github.com/alvin-aldino-rahmatullah/student-finance-app.git
```

Masuk ke folder project:

```bash
cd student-finance-app
cd student-finance-app
```

---

## ⚠️ PENTING

Sebelum mulai ngoding:

```bash
git pull origin main
```

Sebelum push:

```bash
git pull origin main
```

kalau langsung push tanpa pull dulu, nanti bisa conflict.

---

## 🛠️ Cara Push Perubahan

Abis selesai ngoding:

```bash
git add .

git commit -m "Menambahkan fitur login"

git push origin main
```

Commit message jangan:

```bash
git commit -m "fix"
git commit -m "update"
git commit -m "hehe"
```

Biar kita semua masih bisa ngerti siapa ngapain

Contoh yang bagus:

```bash
git commit -m "Menambahkan fitur login"
git commit -m "Menambahkan CRUD transaksi"
git commit -m "Menambahkan statistik bulanan"
```
